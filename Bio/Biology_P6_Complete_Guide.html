What Paper 6 Tests

Paper 6 (Alternative to Practical) tests practical skills without requiring actual lab work. You are given descriptions of experiments, results, graphs, and data — and you answer questions using these. Total: 40 marks. Time: 1 hour.

Typically there are 3 questions, each carrying about 12–15 marks. Questions are based on real experimental scenarios from Biology topics in the 0610 syllabus.

Graph plottingLabel axes fully with units, choose suitable scale using >½ paper, plot as × or dots, join with ruler4–6 marks
CalculationsShow all working, include units, correct rounding, percentage change/magnification/rate1–3 marks

What Examiners Are Really Looking For

Specific biological vocabulary with correct units. Answers grounded in the data provided. Realistic, named improvements. Control variables stated by name — not just "same conditions".

Examiner Priority List

  • Use data in every answer where possible — compare figures, quote measurements
  • Name control variables explicitly — "same volume of solution", "same temperature", "same species of potato"
  • Describe, don't explain unless the command word is "explain"
  • Conclusions must follow from evidence only — don't go beyond what the results show
  • Include units every time in calculations, table headings, and axis labels
  • Graph: both axes fully labelled with units — no labels = no marks
  • Improvements must be practical and named — "use a water bath to maintain a constant temperature of 40 °C"

Paper 6 Exam Strategy

How to Read a Paper 6 Question

  1. Read the experiment description fully before looking at any question
  2. Underline the command word in each sub-question
  3. Check the mark allocation in brackets — it tells you how many points to make
  4. Look at any figures, tables, or diagrams provided and understand them first
  5. If a graph is given, read the axes carefully including units and scale

Time Management

Paper 6 = 40 marks in 60 minutes. That is 1.5 minutes per mark. If a question is worth 4 marks, spend about 6 minutes on it. Do not spend more than 10 minutes on any single question part.

  • Start with what you know — earn easy marks first
  • Tables and graphs take longer — budget 8–10 minutes for these
  • Planning questions need structured thinking — allocate 7–9 minutes for a 6-mark planning answer
Key Formulas

magnification = image size ÷ actual size
actual size = image size ÷ magnification
percentage change = (change ÷ original) × 100
percentage = (part ÷ whole) × 100
mean = total sum ÷ number of readings
rate = quantity ÷ time

Checklists for Each Question Type

Graph Questions ✔

  • Both axes fully labelled with quantity AND units (e.g. "time / s")
  • Scale uses more than half the grid area — not too cramped
  • Scale goes up in even steps (1, 2, 5, 10 etc.)
  • All points plotted as neat small × or dot in the exact right position
  • Points joined by straight ruled lines between them (unless smooth curve is justified)
  • Anomalous points circled but NOT included in the line
  • Title not always required but add it if asked

Table Questions ✔

  • All lines drawn with a ruler
  • Column headings include units in the heading (e.g. "Temperature / °C")
  • No units written in the data cells themselves
  • All values recorded to the same number of decimal places
  • Headings are clear and descriptive

Planning Questions ✔

  • Independent variable named and how it will be varied (range + values)
  • Dependent variable named and how it will be measured (instrument + units)
  • At least two named control variables with how they will be kept constant
  • Method described step-by-step in logical order
  • Repeats mentioned: "repeat each value three times and calculate a mean"
  • Safety consideration included if hazards are present

Evaluation Questions ✔

  • Identify a specific source of error (not just "human error")
  • Explain WHY it causes inaccuracy
  • Suggest a specific, realistic improvement that addresses it
  • Do not suggest changing the basic experiment design

Calculation Questions ✔

  • Write the formula first
  • Substitute values clearly
  • Show all working steps
  • Include the correct unit in the answer
  • Round to an appropriate number of significant figures
When comparing data, always quote actual numbers from the paper. Never say "results increased" — say "results increased from 23 s to 41 s".
Never say "keep everything the same" as a control variable. You must NAME what variable is kept the same.

Analysis of the 6 Provided Examinations

Paper 1: 0610/61/M/J/2025 (May/June 2025) S25 Paper 61
Q1 — Diffusion Investigation (Agar Cubes + HCl)

What tested: Effect of concentration on the rate of diffusion. Agar containing universal indicator (green → red) placed in different concentrations of HCl. Time recorded to change colour completely.

Skills: Table construction (4 marks), conclusion (1 mark), identifying IV/DV (2 marks), two improvements (2 marks), two hazards (1 mark).

What the mark scheme rewards for the table: Ruled lines, headings with units in column head (concentration / mol dm⁻³ and time / s), no units in data cells, all data in correct columns, values converted correctly from min:sec to seconds.

Ideal conclusion: "As the concentration of HCl increases, the time taken for the agar to turn red decreases." — Always state direction and link IV to DV.

Students often write "time decreased" without saying "as concentration increased". The mark requires a comparative statement.

Improvements expected: Repeat each concentration and calculate a mean; use a colorimeter to measure colour change instead of visual judgement; use agar pieces of identical size; control temperature using a water bath.

Hazards: HCl is corrosive — causes skin burns / eye damage. Knife/scalpel is sharp — could cut skin.

Q1(b) — Planning: Effect of Temperature on Osmosis in Potato

What tested: Full planning — IV, DV, controls, method, safety.

IV: Temperature of water bath (e.g. 10, 20, 30, 40, 50 °C)

DV: Change in length/mass of potato strips after a set time

Controls: Same concentration of solution, same initial size of potato strips, same species of potato, same time in solution

State at least 3 named control variables for full marks. Include how you will keep them constant.
This paper tested: table construction with unit conversion, conclusion writing, hazards, and a planning question. Classic Paper 6 structure.
Paper 2: 0610/62/M/J/2025 (May/June 2025) S25 Paper 62
Q1 — Osmosis in Potato Discs with Salt Solutions

What tested: Effect of salt solution concentration on osmosis in potato tissue. Potato discs placed in 0.0, 0.5, 1.0 mol dm⁻³ salt solutions. Change in total length measured.

Skills: Calculating volumes for dilution (1 mark), table construction, percentage change calculation, graph plotting, conclusion, evaluation.

Dilution calculation: To make 0.5 mol dm⁻³ from 1.0 mol dm⁻³ in 50 cm³ total — need 25 cm³ of 1.0 mol dm⁻³ + 25 cm³ distilled water.

Percentage change = (final length − initial length) ÷ initial length × 100

Graph expectations: Line graph, concentration on x-axis, percentage change in length on y-axis. Full axis labels with units. Points joined with ruled lines. Anomalous point if present should be circled.

The examiner gives marks for: scale using over half the grid, even scale intervals, fully labelled axes with units, all points plotted correctly as ×, ruled lines between points.
Q1 Continued — Conclusion and Evaluation

Ideal conclusion: "As the concentration of salt solution increases, the percentage decrease in length of potato discs increases. This is because water moves out of the potato cells by osmosis down a water potential gradient into the more concentrated salt solution."

Do NOT say "the potato gained water" if the data shows it decreased in length. Always base conclusions on the given data.
Paper 3: 0610/63/M/J/2025 (May/June 2025) S25 Paper 63
Q1 — Effect of Temperature on Catalase Activity

What tested: Catalase (from potato extract) breaks down hydrogen peroxide → water + oxygen foam. Student measures height of foam at cold (C), room temperature (R), and hot (H).

Skills: Table construction with measurements from diagram, conclusion, identifying anomalies, evaluation.

Table requirements: Three rows (C, R, H), headings: "test-tube" and "height of foam / mm", drawn with ruler, units in heading only.

Expected trend: Foam increases from cold to room temperature (enzyme more active), then decreases at very high temperature (enzyme denatures).

For the conclusion — state the specific test-tube letters, refer to foam height values, and link to enzyme activity. "The height of foam was greatest in test-tube R (room temperature), showing catalase is most active at room temperature."
Q2 — Enzyme Invertase Investigation (Yeast)

Context: Yeast invertase breaks down sucrose into reducing sugars. Two concentrations of invertase extract (10%, 100%) tested with Benedict's solution.

Expected results: 100% invertase → brick-red (large amount of reducing sugar), 10% invertase → green/yellow (smaller amount), Control (water) → blue (no change).

Table for W25/63: Three columns: test-tube (A, B, W), colour of liquid, conclusion (reducing sugars present/absent).

For Benedict's results: always state the exact colour seen — "brick-red" not just "red". "Blue" = negative, "green/yellow/orange/brick-red" = positive reducing sugar, intensity indicates relative amount.
Paper 4: 0610/61/O/N/2025 (Oct/Nov 2025) W25 Paper 61
Q1 — Transpiration in Celery (Effect of Temperature)

Context: Celery stalks placed in blue dye at cold (4 °C) and warm (40 °C) temperatures. Sections cut from stalk to measure how far dye has moved up the xylem.

Skills: Temperature change calculation (1 mark), table construction (4 marks), conclusions, drawing labelled diagram of xylem, evaluation.

Temperature calculation: Change = final temp − initial temp (include sign: positive if increased, negative if decreased).

Table requirements: Headings: water-bath (C or W), number of 5 mm sections, total distance moved by blue dye / mm. Units in headings.

Expected conclusion: Water (dye) moved further up the xylem at higher temperature — transpiration rate is greater at higher temperatures because water molecules have more kinetic energy.

Drawing of xylem cross-section: use pencil, draw with clear continuous lines, NO shading, label xylem tissue clearly with a line that clearly points to the structure.
Q2 — Microscopy and Magnification

What tested: Biological drawing from a microscope image + magnification calculation.

magnification = image size ÷ actual size
actual size = image size ÷ magnification
Always measure image size in mm first, then convert to µm if actual size is given in µm (1 mm = 1000 µm).
Paper 5: 0610/62/O/N/2025 (Oct/Nov 2025) W25 Paper 62
Q1 — Yeast Respiration with Different Glucose Concentrations

Context: Yeast ferments glucose solutions (1% and 4%). Carbon dioxide produced is collected in an inverted test-tube in water. Volume of gas measured over time.

Skills: Understanding dilution (1% from 5% glucose), table + graph construction, rate calculation, conclusion, evaluation.

Rate calculation: Rate = volume of gas produced ÷ time taken (cm³/min).

Graph: Time (x-axis) vs volume of gas / cm³ (y-axis). Two lines on same axes — label each line.

Conclusion: "As the concentration of glucose increases, the rate of CO₂ production increases, showing that yeast respires faster when more substrate (glucose) is available."

When two lines are plotted, compare them using actual data values. State which is greater and by how much.
Paper 6: 0610/63/O/N/2025 (Oct/Nov 2025) W25 Paper 63
Q1 — Invertase Enzyme Investigation

Context: Invertase enzyme (from yeast) breaks sucrose into reducing sugars. Two concentrations (10%, 100%) + control (distilled water). Benedict's test used to detect reducing sugars.

Table: Test-tube, colour of Benedict's solution, conclusion (reducing sugar present / absent / trace).

Conclusion: "The higher the concentration of invertase, the more sucrose was broken down into reducing sugars, as shown by the more intense colour change in Benedict's solution."

Q2 — Planning and Evaluation Question

Skills: Full planning with named variables, improvement suggestions, linking results to biological concepts (enzyme concentration → reaction rate → product formation).

This paper reinforced the pattern: enzyme investigation in Q1 (food test + table), planning or evaluation in Q2, and often a transport/ecology data question in Q3.

Chapter-by-Chapter Practical Skills Guide

Chapter 1
Characteristics of Living Organisms

P6 link: Rarely directly tested as an experiment, but used in evaluation and observation questions. Identifying MRSGREN characteristics from data.

If asked to observe an organism, describe its characteristics using precise scientific language (nutrition, respiration, movement, growth, excretion, reproduction, sensitivity).

Likely question: Describe what observations suggest the organism is alive (e.g. movement, gas exchange, growth).

Chapter 2
Organisation of Organisms

P6 link: Microscopy, biological drawing, measuring cells, magnification calculations.

Skills: Drawing cells accurately, calculating magnification from scale bar or actual size, identifying organelles.

magnification = image size ÷ actual size
Never shade a biological drawing. Use clear, sharp, continuous pencil lines. No sketch lines.
Chapter 3
Movement In & Out of Cells

P6 link: Very heavily tested. Osmosis in potato/celery, diffusion in agar, effect of concentration on movement rate.

Common experiments: Potato discs in salt/sugar solutions (mass or length change), agar cubes in HCl, dialysis tubing osmosis.

IV: Concentration of solution. DV: Change in mass / length / time taken.

Controls: Same initial mass/length, same temperature, same time, same species of potato, same volume of solution.

Percentage change is almost always required: (final − initial) ÷ initial × 100. Always include the % sign.
Chapter 4
Biological Molecules

P6 link: Food tests — core, heavily tested.

MoleculeTestPositive Result

ProteinBiuret (NaOH + CuSO₄)Blue → violet/purple

Always state the actual colour seen — NOT "colour change" or "turns positive". Write "turns brick-red" not "positive result".
Chapter 5
Enzymes

P6 link: Very heavily tested. Enzyme activity investigations appear in multiple papers.

Common experiments: Catalase + H₂O₂ (foam height), lipase + milk (colour change using thymolphthalein), invertase + sucrose (Benedict's test), amylase + starch (iodine test).

IV options: Temperature, pH, enzyme concentration, substrate concentration.

DV options: Time for colour to disappear, rate of reaction, foam height, volume of gas produced.

Graph of enzyme activity vs temperature: rises to optimum, then drops sharply. Enzyme is DENATURED at high temperatures (not "killed").
Never say the enzyme is "killed" by heat. Say it is "denatured" — the active site changes shape so substrate cannot bind.
Chapter 6
Plant Nutrition

P6 link: Photosynthesis investigations — light intensity, CO₂ concentration, temperature; starch testing; leaf structure drawings.

Common experiments: Aquatic plant (Elodea) bubble counting at different light intensities; starch test on variegated/destarched leaves; hydrogencarbonate indicator colour changes.

IV: Distance of lamp from plant (→ changes light intensity). DV: Number of bubbles per minute.

Light intensity is proportional to 1/distance². But in P6 you use distance as the IV — do NOT say intensity was the IV unless the question states this.
For the starch test on a leaf — state colour of each region precisely: "green area turned blue-black (starch present), white area remained brown/yellow (no starch)".
Chapter 7
Human Nutrition

P6 link: Food test practicals, dietary analysis, identifying nutrients from experimental data.

Likely P6 format: A table of food test results is given — interpret which nutrients are present. Or plan a method to test for a named nutrient.

When asked to interpret food test results, state what each result shows before giving a conclusion. Don't jump straight to "the food contains starch".
Chapter 8
Transport in Plants

P6 link: Transpiration (potometer), water movement in xylem (celery/dye), stomata observation, rate calculations.

Key skills: Rate = distance moved by bubble ÷ time (mm/min) in potometer; drawing cross-section of stem; counting stomata per field of view.

IV (potometer): Wind speed, temperature, humidity, light intensity. DV: Rate of movement of air bubble.

Rate in potometer = distance / time. If cross-sectional area of tube is given, volume = area × distance.
Chapter 9
Transport in Animals

P6 link: Heart rate investigation, effect of exercise on pulse rate, graph of heart rate vs time.

Skills: Graph plotting (line graph, time vs heart rate/bpm), rate calculation, evaluation of method for measuring pulse rate.

Typical question: Plot heart rate before, during, and after exercise. Describe the pattern with data. Suggest why heart rate increases during exercise.

When describing a graph trend: state the starting value, direction of change, any peak, and the final value. Always include numbers.
Chapter 10
Diseases and Immunity

P6 link: Rarely a direct practical, but may appear as data interpretation — antibody levels over time, infection rates in populations, evaluating vaccine trial data.

Likely question: Graph of antibody titre over time — describe the pattern, compare primary/secondary immune response using data.

When interpreting immunity graphs, describe the lag period, the peak, and the rate of decline using actual values from the graph.
Chapter 11
Gas Exchange and Respiration

P6 link: Frequently tested. Yeast respiration investigations, limewater experiments, breathing rate, hydrogencarbonate indicator colour changes.

Common experiments: Yeast + glucose at different temperatures (bubble counting), limewater to compare CO₂ in inspired vs expired air, hydrogencarbonate indicator in light/dark.

Graph: Rate of respiration (bubbles/min) vs temperature — bell-shaped curve with optimum.

Aerobic respiration: glucose + O₂ → CO₂ + water (+ energy). Anaerobic (yeast): glucose → ethanol + CO₂. Do not confuse the two when writing conclusions.
Chapter 12
Coordination and Response

P6 link: Reaction time investigations, sensory response experiments, tropism investigations.

Typical practical: Ruler-drop reaction time test; measuring effect of stimulus intensity on response; geotropism or phototropism in seedlings.

Skills: Table with repeat readings + mean, anomaly identification, graph of response vs stimulus strength.

Chapter 13
Excretion and Homeostasis

P6 link: Less commonly a direct practical; may appear as data interpretation of temperature regulation, urine concentration data, or blood glucose levels.

Likely question: Graph of blood glucose over time — describe the pattern, identify the peak, explain the role of insulin.

When interpreting homeostasis data — always describe the mechanism (negative feedback) and name the hormone involved.
Chapter 14
Reproduction in Plants

P6 link: Germination investigations, pollen tube growth, effect of conditions on germination rate.

Common experiment: Seeds in different conditions (temperature, light, water availability) — count % germination over time.

IV: Temperature / light / water. DV: Percentage of seeds germinated.

Chapter 15
Reproduction in Humans

P6 link: Interpreting graphs of hormone levels during menstrual cycle, fertility data, pregnancy development data.

Likely question: Line graph of oestrogen/progesterone over 28 days — describe patterns, identify ovulation, compare hormone levels using data.

Chapter 16
Chromosomes, Genes and Proteins

P6 link: Rarely a direct practical. May appear as analysis of genetic variation data or gel electrophoresis interpretation.

Likely question: Interpreting band patterns in gel electrophoresis, drawing conclusions about genetic relationships.

Chapter 17
Variation and Selection

P6 link: Measuring continuous variation in a population, plotting histograms, calculating mean and range.

Common experiment (ES 17.1): Measuring finger length in a class — record in a frequency table, construct a histogram.

Skills: Frequency table, histogram (bars touching), mean calculation, mode identification.

Histograms have BARS THAT TOUCH — this shows continuous data. Bar charts have gaps between bars for discrete data.
Chapter 18
Organisms and Their Environment

P6 link: Very frequently tested. Quadrat sampling, transects, population data analysis, food webs, ecological investigations.

Skills: Calculating population density from quadrat data, percentage cover, interpreting transect diagrams, bar charts of species abundance.

Formula: Population estimate = (number in sample ÷ area of quadrat) × total area.

For quadrat questions — always state how many quadrats were used and how the area was randomly selected to reduce sampling bias.
Chapter 19
Human Influences on Ecosystems

P6 link: Interpreting pollution data, biodiversity indices, deforestation impact data, eutrophication graphs.

Likely question: Graph of species diversity vs pollution level — describe the pattern, explain the trend, evaluate the investigation method.

Chapter 20
Biotechnology & Genetic Modification

P6 link: Biological washing powder investigation (Exp. Skills 20.1), fermentation data analysis, enzyme function in industrial context.

Planning skills: Planning how to compare biological vs non-biological washing powders — stain removal, temperature, IV/DV/controls.

For biotechnology planning: IV = type/concentration of enzyme preparation. DV = degree of stain removal (measured by eye or colorimeter). Controls = same fabric, same stain, same temperature, same time.

Complete Experimental Skills Master Guide

Observation & Identification

Describe what you see using precise scientific language. State exact colours, sizes, shapes. Never use vague terms like "it changed".

Always state: what changed, in which direction, and by how much (if measurable).
Measuring with Apparatus

Use a ruler to measure in mm/cm. Use a measuring cylinder or syringe for volumes (cm³). Use a thermometer for temperature (°C). Record to the precision of the instrument.

Read volume from the bottom of the meniscus. Eye must be level with the meniscus.
Recording in Tables
  • Draw all lines with a ruler
  • Units in column headings only — use "/" notation: "Temperature / °C"
  • No units repeated in data cells
  • All values to consistent decimal places in same column
  • IV in first column, DV in subsequent columns
Graphing
  • IV → x-axis; DV → y-axis
  • Both axes labelled with quantity and unit
  • Scale: even intervals, uses >½ of paper
  • Plot as neat × or dot
  • Join points with straight ruled lines (unless told otherwise)
  • Anomalous points: circle them, do not include in line
Calculations
  • Write formula first
  • Substitute values with units
  • Show all working
  • Include unit in final answer
  • Round appropriately (same decimal places as data)
Microscopy & Biological Drawing
  • Use sharp pencil
  • Single continuous lines — no sketch marks
  • No shading
  • Labels drawn with horizontal lines (not diagonal)
  • Drawing fills the space provided
  • Title and magnification stated
magnification = image size ÷ actual size
Identifying Variables

Independent (IV): The variable you change. State what it is and the range of values you will use.

Dependent (DV): The variable you measure. State what it is and how you will measure it.

Control variables: Name at least 2–3 specific variables that could affect the DV and explain how you will keep each constant.

Planning Investigations
  1. State the aim
  2. Name IV and range of values
  3. Name DV and how measured
  4. Name and explain control variables
  5. Write clear, logical method steps
  6. State number of repeats
  7. Note safety considerations
  8. Draw results table outline
Reliability, Accuracy & Precision

Reliability: Results you can trust — achieved by repeating and taking means.

Accuracy: How close to the true value — improved by using calibrated instruments, removing systematic errors.

Precision: How repeatable — more decimal places, smaller scale divisions.

These three terms mean different things. Using one when you mean another loses marks.
Evaluation & Improvements

Identify a specific source of error → explain the problem it causes → suggest a named improvement that would reduce it.

Example: "The temperature of the water bath may have fluctuated during the experiment, affecting enzyme activity. Using an electronic water bath that maintains a constant temperature would improve accuracy."

Identifying Anomalies

An anomalous result does not fit the general trend of the data. On a graph, circle the anomalous point and do not include it in the line of best fit. State that the result was excluded and why.

To decide if a result is anomalous: compare it to the trend of all other results. If it is significantly different from the trend, it is anomalous.
Drawing Conclusions

A conclusion must: state the relationship between IV and DV; include specific data values; not go beyond what the evidence shows; use appropriate biological vocabulary.

Structure: "As [IV] increases/decreases, [DV] increases/decreases. For example, when [IV] was [value], [DV] was [value], compared to [value] when [IV] was [value]."

💧 Osmosis Investigation (Potato/Plant Tissue)

  • Aim: Investigate effect of solution concentration on osmosis in potato tissue
  • IV: Concentration of salt/sugar solution (e.g. 0.0, 0.25, 0.5, 0.75, 1.0 mol dm⁻³)
  • DV: Change in mass or length of potato strip/disc (%)
  • Controls: Same initial mass/length, same potato species, same temperature, same time in solution, same surface area of potato, same volume of solution
  • Graph: Line graph. Concentration (x-axis) vs percentage change in mass (y-axis). The line should cross zero at the point where water potential of solution = water potential of potato cells (isotonic point)
Calculate percentage change: (final − initial) ÷ initial × 100. Use + for gain, − for loss. Plot % change, not absolute change, to allow fair comparison.
Conclusion must use the word "osmosis" and explain direction of water movement. "Water moved out of potato cells by osmosis into the more concentrated salt solution, causing the potato to decrease in mass."

⚗️ Enzyme Activity Investigations

  • Catalase + H₂O₂: Foam height indicates activity. Higher temperature → more activity (to optimum), then drops (denaturation).
  • Lipase + milk: Thymolphthalein indicator (blue → colourless when fatty acids produced). Time for colour to disappear = DV.
  • Invertase + sucrose: Benedict's test detects reducing sugars produced. Brick-red = high enzyme activity.
  • Amylase + starch: Iodine test at intervals. Starch breaks down → iodine stays yellow/brown.
For rate calculations: rate = 1 ÷ time (if DV is time taken). Or rate = amount of product ÷ time.
Always explain denaturation correctly: "at high temperatures, the bonds holding the enzyme's tertiary structure break, changing the shape of the active site, so the substrate can no longer bind."

🌿 Photosynthesis Investigations

  • Light intensity (Elodea): IV = distance from lamp (cm). DV = bubbles per minute / volume of O₂ per minute. Closer lamp → more bubbles.
  • Controls: Same plant, same temperature (heat shield!), same CO₂ concentration, same time interval for counting.
  • Starch test (variegated leaf): Green areas → blue-black (starch present). White areas → brown (no chlorophyll, no starch).
  • CO₂ need (destarched plant): Leaf with CO₂ → starch present. Leaf without CO₂ (KOH absorbs CO₂) → no starch.
The heat shield in light intensity experiments controls temperature — an essential control variable. If omitted, you cannot conclude that light intensity is the only variable affecting rate.

🫁 Respiration Investigations

  • Yeast + glucose (temperature effect): IV = temperature. DV = bubbles/min (CO₂ produced). Use at least 5 temperatures. Graph = bell curve.
  • Inspired vs expired air (limewater): Breathe in → through limewater → breathe out → through second limewater. Compare cloudiness of limewater. Expired air = more cloudy = more CO₂.
  • Aquatic plant (hydrogencarbonate indicator): In light → photosynthesis > respiration → CO₂ absorbed → indicator turns purple. In dark → only respiration → CO₂ released → indicator turns yellow.
Hydrogencarbonate indicator colours: purple (CO₂ removed), red (normal CO₂ level), yellow (CO₂ increased). Always describe exact colour seen.

🌱 Transpiration (Potometer)

  • Aim: Measure rate of water uptake (∝ transpiration rate) under different conditions
  • IV: Wind speed (fan), temperature, humidity, light intensity
  • DV: Distance moved by air bubble in capillary tube per unit time (mm/min)
  • Controls: Same plant species, same leaf area, same length of capillary tube, same starting position of bubble
  • Rate formula: Rate = distance ÷ time (mm/min)
The potometer measures water UPTAKE, not directly water LOSS (transpiration). They are nearly equal because water lost = water taken up. State this distinction if asked.

🌰 Germination Investigation

  • Conditions tested: Temperature (warm vs cold), light (light vs dark), water (moist vs dry), oxygen (air vs no air)
  • DV: Percentage of seeds that germinate
  • Result: Seeds need warmth, water, and oxygen to germinate. Light is not essential for most seeds.
Always use percentage germination (not raw count) to allow fair comparison across groups with different total seed numbers.

🌍 Ecological Sampling

  • Quadrats: Randomly placed square frames used to estimate percentage cover or population density.
  • Transects: A line along which quadrats or observations are made — used to study how species change along an environmental gradient.
  • Population estimate: (mean number per quadrat × total area) ÷ quadrat area
  • Random sampling: Use random coordinates (e.g. from random number tables) to place quadrats — reduces bias.
For evaluation of sampling: state that more quadrats should be used to improve reliability; or that quadrats should be placed more randomly to reduce bias.

Command Words

state / name / identify

What examiner wants: A single clear answer. No explanation needed.

Common mistake: Adding unnecessary explanation — you won't gain extra marks and may contradict yourself.

Example: "State the independent variable." → "The concentration of hydrochloric acid."

describe

What examiner wants: What you observe or what happens — with data. Not WHY it happens.

Common mistake: Explaining the biology when the question only asks to describe. Loses time.

Example: "Describe the results." → "As concentration increased from 0.4 to 1.0 mol dm⁻³, the time decreased from 145 s to 62 s."

explain

What examiner wants: The biological reason WHY something happens. Use because / so that / therefore.

Common mistake: Just re-describing the results without giving the mechanism.

Example: "Explain why the potato decreased in length." → "Water moved out of potato cells by osmosis because the salt solution had a lower water potential than the cell contents."

compare

What examiner wants: State the similarity AND/OR difference. Must refer to BOTH items being compared, with data.

Common mistake: Describing each separately instead of making a direct comparison.

Example: "Test-tube A showed a brick-red colour whereas test-tube B remained blue, showing that A contained reducing sugar but B did not."

suggest

What examiner wants: A reasonable scientific explanation that may not be directly in the data. Your thinking is being tested.

Common mistake: Writing a fact from memory that doesn't relate to the context given.

Example: "Suggest why the time taken at 0 °C was very long." → "At low temperatures, enzyme molecules have less kinetic energy, so the frequency of enzyme-substrate collisions is reduced, slowing the reaction rate."

predict

What examiner wants: What you expect the result to be, with a brief scientific justification.

Example: "Predict the result for 2.0 mol dm⁻³." → "The time would be shorter than at 1.0 mol dm⁻³ because higher concentration increases diffusion rate."

calculate

What examiner wants: Show formula → substitute values → give numerical answer with unit.

Common mistake: No unit in the answer, or no working shown.

determine

What examiner wants: Find a value from a graph or data (often by reading the graph or performing a calculation from it).

Example: "Determine the rate of reaction at 30 °C from your graph." → Read value from y-axis at x = 30.

draw / plot

Draw: Make a biological drawing or sketch diagram.
Plot: Mark data points on a graph and connect them.

Common mistake for plot: Not labelling axes, using wrong scale, not joining points with ruler.

construct

What examiner wants: Build a table or draw a results chart from scratch. All lines with ruler, correct headings with units.

conclude

What examiner wants: A statement of what the results show — must be based only on the evidence. Link IV to DV. Do not go beyond the data.

Common mistake: Drawing a conclusion that goes further than the evidence (e.g. claiming to prove a hypothesis from one experiment).

evaluate

What examiner wants: Assess the quality of the method/experiment. Identify weaknesses. Suggest improvements. Assess reliability of results.

The Most Common Paper 6 Mistakes

#MistakeCorrect Approach
1Missing units in calculations or table headingsEvery measurement needs a unit. In tables: "Time / s" not "Time". In answers: "4.5 mm" not "4.5".
2Wrong graph type — bar chart instead of line graphUse LINE graph when IV is continuous (temperature, concentration, time). Use bar chart when IV is categoric (species, type of food).
3Axes not labelled or units missing from axesBoth axes must have: quantity name + "/" + unit. E.g. "concentration / mol dm⁻³"
4Graph not drawn with a ruler (freehand lines)All straight lines between data points MUST be drawn with a ruler.
5Vague control variable: "same conditions"Name the specific variable: "same temperature maintained at 25 °C using a water bath"
6Comparing without numbersNever say "the results were higher". Say "the rate increased from 12 to 34 bubbles/min".
7Explaining instead of describingIf the command word is "describe", state what happened. If "explain", then give the reason.
8Not showing working in calculationsAlways write formula first, then substitute, then calculate. Even if final answer is wrong, you can get method marks.
9Wrong rounding or significant figuresRound to the same number of decimal places as the data provided. Check significant figures match the precision of the measurement.
10Misreading scales on graphs/measuring cylindersAlways read from the bottom of the meniscus. For graph scales, check the value of each small division before reading.
11Shading in biological drawingsNEVER shade. Use single, clean lines. Shading = zero marks for drawing quality.
12Unrealistic improvements (e.g. "use a better thermometer")Improvements must be specific and practical: "use an electronically controlled water bath to maintain temperature at ±0.1 °C"
13Conclusion goes beyond dataConclude only from the evidence in the paper. Don't assume results would extend beyond the range tested.
14Confusing accuracy, precision, reliabilityReliability = repeatable. Accuracy = close to true value. Precision = many decimal places / small intervals.
15Describing food test as "positive" without stating colourAlways state the actual colour: "turned brick-red" not "positive result"
16Scale not using more than half the graph paperChoose a scale so that all your data points use at least 50% (ideally 75%) of the available grid.
17Identifying "human error" as a source of errorBe specific: "difficulty judging when the colour had completely changed by eye" is better than "human error".
18Planning with only 1 control variableAlways name at least 2–3 control variables with specific methods for keeping them constant.

Quick Revision Summary Sheets

📐 Graph Rules
  • IV on x-axis, DV on y-axis
  • Both axes labelled: "quantity / unit"
  • Even scale, uses >½ grid
  • Points plotted as neat ×
  • Straight ruled lines between points
  • Anomalies circled, excluded from line
  • Title if asked
📋 Table Rules
  • All lines drawn with ruler
  • Headings with units (e.g. "Time / s")
  • No units in data cells
  • Consistent decimal places per column
  • IV in first column
  • Include a mean column if repeats present
🔢 Formula Sheet
  • magnification = image ÷ actual
  • actual = image ÷ magnification
  • % change = (change ÷ original) × 100
  • % = (part ÷ whole) × 100
  • mean = total ÷ n
  • rate = quantity ÷ time
  • rate = 1 ÷ time
📝 Planning Template
  1. Aim: To investigate the effect of [IV] on [DV]
  2. IV: [name], varied by [method], values: [list]
  3. DV: [name], measured by [instrument] in [unit]
  4. Controls: [name 3, say how kept constant]
  5. Method: Set up → do experiment → measure → repeat
  6. Repeats: Repeat 3 times, calculate mean
  7. Safety: [name hazard and precaution]
⚗️ Food Tests Quick Ref
  • Starch: Iodine → blue-black
  • Reducing sugar: Benedict's + heat → brick-red
  • Protein: Biuret → violet
  • Lipid: Ethanol emulsion → cloudy white
  • Vitamin C: DCPIP → colourless
✏️ Biological Drawing Rules
  • Use sharp pencil only
  • Single, continuous lines
  • No shading
  • Horizontal label lines with ruler
  • Drawing fills the space
  • Title + magnification stated
💬 Sentence Starters

Conclusion: "As [IV] increases, [DV] increases/decreases. For example…"

Comparison: "[A] was greater than [B] — [value A] compared to [value B]"

Description: "The results show that as [IV] changed from [v1] to [v2], [DV] changed from [v1] to [v2]."

Improvement: "[Problem identified] — this could be improved by [specific named method]."
